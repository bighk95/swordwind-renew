# Swordwind-renew

칼바람 나락 플레이어별 기여도를 산정하여 순위를 책정합니다.

## 기능명세

### 메인페이지
- 플레어어의 ID, TAG를 입력받아서 검색을 실행합니다.
- 실력차이가 많이 나는 경우, 좌하단의 밸런스 패치 모달을 통해서 페널티, 어드밴티지를 임의대로 적용할 수 있습니다.
